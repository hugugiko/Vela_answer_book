## 快速开始
推荐使用官方AIoT IDE打开本项目 方法如下  
请确保AIoT IDE正确被配置  
```
1 创建一个新的项目文件夹
2 放入src文件夹
3 使用AIoT IDE打开项目(!请打开一开始创建的项目文件夹，不要直接打开src文件夹)
4 右侧等待自动补全项目文件
```
